### 풀이 방법
출력
미로를 탈출하는데 드는 이동 횟수의 최솟값을 출력
민식이가 탈출할 수 없다면 -1을 출력

조건
빈칸은 이동할 수 있음
벽은 이동할 수 없음
열쇠는 이동할 수 있고 집을 수 있다.
문은 대응하는 열쇠가 있을 때만 이동가능

현재 서있는 곳은 0
달이 있는 곳은 1 -> 도착하면 미로를 탈출

패턴
이동 횟수의 최솟값을 구하기 위해서 BFS로 구현

문에 대응하는 열쇠를 맞추기 위해서 어떤 식으로 이동하는가?

방문배열없이 이동횟수를 제한하지 않고 진행해야 할 것
